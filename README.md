# FLACNameChanger
 Batch file renamer <br>
 Tool designed for my personal use, but modifiable for other tasks.<br>
# What does it do?
 This tool is made to set all tracks to a cetrain standard (in this case {track no. (00 format)} - {song name}{extension}) <br>
 The logic is not really dynamic so it *has* to be set at the artist folder.<br>
 It then searches each subfolder (which are albums) and in them renames the files with .flac and .mp3 extensions that are named with different numbering scheme and artist name.
